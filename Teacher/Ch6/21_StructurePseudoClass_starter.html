:root selects a document's root element. For an HTML document, it will always select the HTML element.
:empty selects an element of type E if it has no children or text content.
:target selects an element of type E if it is the target of a referring URL. The :target pseudo selector in CSS matches when the hash in the URL and the id of an element are the same
:not(s) selects any element which does not match the selector strings.
