Text Pseudo-Elements

Text-based pseudo-elements match elements that are not easily identifiable in the document tree for a page.

The following list describes some of the more commonly used pseudo-elements. Notice that in a CSS rule you use a double colon to concatenate them to a text item.

  • :first-letter 選第一行第一個字
  • :first-line 選第一行
  • :before pseudo-element新增content到element之前
  • :selection 使用者選取的文字
